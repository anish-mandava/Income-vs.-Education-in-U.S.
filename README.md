This project investigates the relationship between educational attainment and median income across the United States using data from the U.S. Census Bureau (ACS 2024).

We analyze both state-level and county-level data to answer:

Does higher education correspond to higher income across U.S. states?
How strong is this relationship?
Which counties have the highest median income within each state?

The results show a strong positive correlation between education and income, supported by statistical modeling and visualization.
